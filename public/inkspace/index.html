Guide d'utilisation

InkSpace v3.0
Documentation officielle — Guide utilisateur complet

🎨 Outils de dessin libre

PStylo — Tracé précis et fin. Idéal pour les annotations et schémas détaillés.
BPinceau — Effet brosse organique avec dispersion. Pour un rendu artistique.
MMarqueur — Surligneur semi-transparent épais. Parfait pour mettre en évidence.
GCalligraphie — Trait sensible à la direction du mouvement. Écriture élégante.
EGomme — Efface les objets dessinés. Curseur circulaire visible. Ajustez la taille dans la palette.

📐 Formes 2D

RRectangle (contour ou plein) — Cliquer-glisser pour dessiner.
CEllipse/Cercle (contour ou plein) — Cliquer-glisser.
LLigne droite — Du point A au point B.
AFlèche simple ou double — Indique une direction ou relation.
—Triangle, Losange, Parallélogramme, Pentagone, Hexagone, Étoile — Dans la barre d'outils 2D.

🎲 Formes 3D

⬛Cube — Perspective isométrique avec face supérieure et arêtes. Rendu réaliste.
🔺Prisme triangulaire — Triangle extrudé en 3D avec arêtes pointillées.
🥫Cylindre — Corps cylindrique avec ellipses dessus/dessous. Lignes pointillées pour les arêtes cachées.

〜 Courbe de Bézier

①Sélectionner l'outil Bézier dans la barre.
②Cliquer 4 fois : départ → contrôle 1 → contrôle 2 → arrivée.
③La courbe est tracée automatiquement au 4ème clic.

T Texte

TSélectionner l'outil Texte, cliquer sur le canvas. Saisir le texte, valider avec Ctrl+Entrée ou le bouton OK.
—La taille du texte suit le réglage de taille dans la palette. Multi-lignes supporté.

fx Formules Mathématiques

FSélectionner l'outil Formule, cliquer sur le canvas. La boîte de saisie apparaît.
—Utiliser la notation LaTeX simplifiée : \frac{a}{b}, \sqrt{x}, \alpha
—Les boutons de raccourcis insèrent automatiquement les symboles courants.
—Les fractions \frac{} sont rendu avec numérateur/dénominateur visuellement séparés.

∠ Outil Angle

QActiver l'outil Angle.
①1er clic : place le sommet de l'angle.
②2ème clic : extrémité du premier bras.
③3ème clic : extrémité du second bras. La mesure en degrés s'affiche automatiquement.

⬤ Graphes (orientés & non orientés)

①Ouvrir le panneau Graphes (en-tête).
②Choisir Nœud → cliquer pour placer. Les nœuds reçoivent des étiquettes A, B, C... automatiquement.
③Choisir Arête → cliquer le nœud source, puis le nœud cible.
④Activer Orienté pour les digraphes (flèches). Sinon : arête bidirectionnelle.
⑤Ajouter un poids optionnel à chaque arête.
⑥Déplacer les nœuds avec l'outil Sélectionner — les arêtes se mettent à jour automatiquement.

⬚ Sélection & Déplacement

SActiver l'outil Sélectionner.
—Cliquer sur un élément pour le sélectionner. Le glisser pour le déplacer.
—Cliquer-glisser sur un espace vide pour sélectionner plusieurs éléments.
DelSupprimer les éléments sélectionnés.
Ctrl+ATout sélectionner.

✋ Navigation sur le canvas

VOutil Main — Déplacer tout le canvas (contenu + grille simultanément).
EspaceMaintenir Espace = activer la main temporairement.
MoletteZoom centré sur le curseur (contenu + grille).
+/-Zoom avant/arrière.
⌂Bouton Reset zoom — revient à 100%, centré.
